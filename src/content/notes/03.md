勉強会で Claude Code の並列実装について知って、やってみたくなり、wtp / lazygit / ghostty を取り入れてみました。

wtp はワークツリーの分離がすごく簡単で、ブランチごとに環境を切り替えるのがスムーズでした。ghostty も見やすくて操作しやすく、複数タブを行き来する作業と相性が良かったです。

並列で Claude Code を動かす体験自体は新鮮でした。ただ、複数の作業を同時に進めるのは思った以上に疲れて、競合も避けたいのでまだ模索中です。

ツール自体もまだ慣れようとしているところ。しばらく使い続けて、自分に合うリズムを見つけていきたいです。
