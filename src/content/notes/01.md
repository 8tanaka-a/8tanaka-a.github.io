このポートフォリオを Astro で作りました。
SSG で完結できて、コンポーネント指向で書けるフレームワークを探していました。

基本は静的に出しつつ、必要な部分だけ CSR や SSR に切り替えられる作り。
必要な分だけ JavaScript を載せるので、ビルドも表示も軽く感じます。

View Transitions が標準で使えるので
CSS に1行足すだけでページ間のアニメーションが繋がる。想像以上に手軽でした。

Content Collections も Markdown に型が付くので、
frontmatter の書き間違いをビルド時に拾えます。

静的サイトを作るなら、しばらくこれで続けてみたいと思っています。
