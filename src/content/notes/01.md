このポートフォリオを Astro で作りました。
SSG で完結できて、コンポーネント指向で書けるフレームワークを探していて行き着きました。

基本は静的に吐き出しつつ、必要な部分だけ CSR や SSR に切り替えられる。
全部を JavaScript で動かす前提じゃないから、ビルドも表示も速いです。

View Transitions が標準で使えるので
CSS に1行足すだけでページ間のアニメーションが繋がる。想像以上に手軽でした。

Content Collections もMarkdown に型が付くので、
frontmatter の書き間違いをビルド時に拾えます。

静的サイトを作るなら、しばらくこれで行きたいと思っています。
